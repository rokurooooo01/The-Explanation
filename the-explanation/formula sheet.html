00 The Guaranteed Question — t-Distribution Derivation Every Paper

Appears in every single end-semester paper from 2019–2025. The form changes slightly but the structure is identical each time. Master this derivation.

1
Given X, Y, U, V, W ~ iid N(μ, σ²). Note that X − Y ~ N(0, 2σ²), so (X−Y)/σ√2 ~ N(0,1).
2
Show that (U − μ)²/σ² ~ χ²(1) for each variable. The expression U² + V² + W² + 3μ² − 2μ(U+V+W) = (U−μ)² + (V−μ)² + (W−μ)² ~ σ²·χ²(3).
3
Apply the definition: t(n) = Z / √(χ²(n)/n) where Z ~ N(0,1) is independent of the chi-squared variable.
4
The constant 'a' in the question is chosen to make the N(0,1) part exactly correct — find it by normalizing (X−Y).
t-definition: T = Z / √(V/n) where Z ~ N(0,1), V ~ χ²(n), independent χ²(n) definition: Sum of n independent squared standard normals Key: (Xᵢ − μ)²/σ² ~ χ²(1), so Σ(Xᵢ−μ)²/σ² ~ χ²(n)
01 Axiomatic Foundations Theory
  • Sample space Ω, events, probability measure P — Kolmogorov's 3 axioms
  • Conditional probability P(A|B) = P(A∩B)/P(B)
  • Independence P(A∩B) = P(A)·P(B); extends to mutual independence
  • Total Probability P(B) = Σ P(B|Aᵢ)·P(Aᵢ) over partition {Aᵢ}
  • Bayes' Theorem P(Aᵢ|B) = P(B|Aᵢ)P(Aᵢ) / Σ P(B|Aⱼ)P(Aⱼ)
Bayes template: Prior × Likelihood / Evidence "If defective, which manufacturer?" → classic exam form
02 Random Variables & Moments Core
  • PMF/PDF, CDF — properties and how to verify validity (integral/sum = 1)
  • E[X] = ∫x·f(x)dx, E[g(X)] = ∫g(x)f(x)dx
  • Var(X) = E[X²] − (E[X])²
  • Moments — nth moment E[Xⁿ], central moments
  • MGF M_X(t) = E[eᵗˣ] — uniquely identifies distribution
  • If MGF known: E[X] = M'(0), E[X²] = M''(0)
Key MGF results to memorise: Binomial: (q + pe^t)^n Poisson: e^(λ(e^t − 1)) Normal: e^(μt + σ²t²/2)
03 Standard Distributions — Complete Reference Heavy Weight
Bernoulli / Binomial
X ~ b(n, p)
P(X=k) = C(n,k)pᵏqⁿ⁻ᵏ
E = np, Var = npq
Sum of Bernoullis
Poisson
X ~ p(λ)
P(X=k) = e⁻λλᵏ/k!
E = λ, Var = λ
E[Xⁿ] = λE[(X+1)ⁿ⁻¹]
Geometric
X ~ g(p)
P(X=k) = qᵏ⁻¹p
E = 1/p, Var = q/p²
Memoryless property
Negative Binomial
r-th success
P(X=k) = C(k−1,r−1)pʳqᵏ⁻ʳ
E = r/p
k ≥ r
Uniform
X ~ U(a,b)
f(x) = 1/(b−a)
E = (a+b)/2
Var = (b−a)²/12
Exponential
f(x) = λe⁻λx
E = 1/λ, Var = 1/λ²
Memoryless: P(X>r+s|X>s)=P(X>r)
Min of exponentials → exponential
Normal
X ~ N(μ, σ²)
aX+b ~ N(aμ+b, a²σ²)
Sum of independents → Normal
Standardise: Z=(X−μ)/σ
Chi-Squared
X ~ χ²(n)
Sum of n squared N(0,1)
E = n, Var = 2n
χ²(m)+χ²(n)=χ²(m+n) if indep.
t-Distribution
T ~ t(n)
Z/√(χ²(n)/n) where Z⊥χ²
Used when σ² unknown
t(∞) → N(0,1)

04 Multivariate Random Variables Core
  • Joint PDF/PMF f(x,y) — integrates/sums to 1
  • Marginal fₓ(x) = ∫f(x,y)dy
  • Conditional f_{X|Y}(x|y) = f(x,y)/f_Y(y)
  • Independence ⟺ f(x,y) = fₓ(x)·f_Y(y)
  • Conditional expectation E[X|Y=y] = ∫x·f_{X|Y}(x|y)dx
  • Covariance Cov(X,Y) = E[XY] − E[X]E[Y]
  • If X,Y independent → Cov = 0 (not converse!)
05 Functions of Random Variables Core
  • CDF method: Find P(Y≤y) = P(g(X)≤y), then differentiate
  • Key result: If X ~ N(0,1), then X² ~ χ²(1)
  • Linear combos: aX+bY: mean = aμₓ+bμ_Y, var = a²σₓ²+b²σ_Y² (if indep.)
  • Sum of normals is normal (reproductive property)
  • MGF method: If Mₓ(t) = M_Y(t) then X,Y same distribution
  • 3D normal distance → scaled chi-squared (missile/particle problems)
If X~N(0,σ²): X²/σ² ~ χ²(1) X²+Y²+Z² (each N(0,σ²)) → σ²·χ²(3)
06 Inequalities Theory
  • Markov's: P(X≥a) ≤ E[X]/a for X≥0, a>0
  • Chebyshev's: P(|X−μ|≥kσ) ≤ 1/k² — needs only mean & variance
  • Equivalently: P(|X−μ|<kσ) ≥ 1 − 1/k²
  • Chebyshev is weaker but universal (no distribution assumed)
  • Exam trick: convert P(a < X < b) → symmetric form around μ, find k = (b−μ)/σ
Schwarz (Cauchy-Schwarz): [E(XY)]² ≤ E(X²)·E(Y²) Chebyshev bound is always ≥ actual probability
07 Limit Theorems Theory
  • WLLN: X̄ₙ →ᵖ μ as n→∞ (converges in probability)
  • Proof via Chebyshev: Var(X̄) = σ²/n → 0
  • CLT: √n(X̄−μ)/σ → N(0,1) for iid finite variance
  • Applies to sums: Sₙ ~ approx N(nμ, nσ²)
  • Use for Binomial (large n), Poisson (large λ), any sum
CLT in practice: P(a < X̄ < b) ≈ Φ((b−μ)/(σ/√n)) − Φ((a−μ)/(σ/√n)) Consistency via WLLN: if E[θ̂]→θ and Var(θ̂)→0 then θ̂ is consistent
08 Point Estimation Statistics
Estimator Properties
  • Unbiased: E[θ̂] = θ
  • Consistent: θ̂ₙ →ᵖ θ as n→∞. Sufficient: E[θ̂]→θ and Var(θ̂)→0
  • Efficiency: smallest variance among unbiased estimators
  • Sample mean X̄ is unbiased + consistent for μ
  • S² = Σ(Xᵢ−X̄)²/(n−1) is unbiased for σ² (note n−1, not n)
Maximum Likelihood Estimation
  • Likelihood L(θ) = Π f(xᵢ; θ)
  • Take log: ℓ(θ) = Σ log f(xᵢ; θ)
  • Solve dℓ/dθ = 0 for θ̂_MLE
  • Poisson MLE: λ̂ = X̄ (sample mean)
  • Normal MLE: μ̂ = X̄, σ̂² = Σ(Xᵢ−X̄)²/n (biased!)
  • Exponential MLE: λ̂ = 1/X̄
09 Confidence Intervals Statistics
  • σ known: X̄ ± z_{α/2} · σ/√n (use Z-table)
  • σ unknown: X̄ ± t_{α/2, n−1} · S/√n (use t-table, df = n−1)
  • 95% CI → α = 0.05 → α/2 = 0.025 → z = 1.96 or t_{0.025, df}
  • 90% CI → α = 0.10 → α/2 = 0.05
  • Sample size: n = (z_{α/2}·σ/E)² for max error E
  • Width = 2 × margin of error
Sampling distribution: X̄ ~ N(μ, σ²/n) exactly if pop. normal (n−1)S²/σ² ~ χ²(n−1) always (for normal population)
10 Hypothesis Testing Statistics
  • H₀ (null) vs H₁ (alternative)
  • Type I error (α): reject H₀ when true
  • Type II error (β): fail to reject H₀ when false
  • Test statistic Z = (X̄ − μ₀)/(σ/√n) for known σ
  • Test statistic T = (X̄ − μ₀)/(S/√n) for unknown σ
  • Chi-squared goodness of fit: χ² = Σ(Oᵢ−Eᵢ)²/Eᵢ, df = k−1−(params estimated)
GoF test: E_i = n·p_i (expected frequencies) df = (classes − 1 − estimated parameters) Reject H₀ if χ²_calc > χ²_{α, df}
