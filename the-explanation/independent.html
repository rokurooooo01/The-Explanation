1. The Independent (Intersecting) Architecture

When analyzing a first-order rational differential equation containing dual linear expressions:

$$\frac{dy}{dx} = \frac{a_1x + b_1y + c_1}{a_2x + b_2y + c_2}$$

If the coefficients fail the parallel test from Chapter 6, the lines are Linearly Independent. This means they are guaranteed to intersect at a specific coordinate point $(h, k)$ in the Cartesian plane.

The Intersecting Test: $$\frac{a_1}{a_2} \neq \frac{b_1}{b_2} \quad \text{or} \quad a_1b_2 - a_2b_1 \neq 0$$

Because these lines intersect away from the origin, the constants $c_1$ and $c_2$ break standard homogeneity. To solve this, we must shift our entire coordinate frame so that the new origin rests exactly on the intersection point, turning the system into a homogeneous equation.

2. Origin Translation Protocol

Execute this systematic structural translation loop:

  1. Find the Intersection: Treat the numerator and denominator as a simultaneous system of equations. Solve for $x=h$ and $y=k$ where: $$a_1h + b_1k + c_1 = 0 \quad \text{and} \quad a_2h + b_2k + c_2 = 0$$
  2. Apply Translation Variables: Introduce new shifted axis variables $u$ and $v$: $$x = u + h \implies dx = du$$ $$y = v + k \implies dy = dv$$
  3. Substitute and Flatten Constants: Plug these into the original DE. The constants $c_1$ and $c_2$ will algebraically cancel out, yielding a Homogeneous Equation: $$\frac{dv}{du} = \frac{a_1u + b_1v}{a_2u + b_2v}$$
  4. Standard Homogeneous Solution: Substitute $v = zu$ (where $dv = z\,du + u\,dz$) to separate variables and solve. Then back-substitute to return to original coordinates.

3. Comprehensive Lecture Example

Let's track a full translation and reduction loop on an intersecting system.

Problem Dossier: Solve $\frac{dy}{dx} = \frac{x + y - 3}{x - y - 1}$.

Step 1: Locate Intersection Point $(h,k)$.
Set up the linear system from the boundaries:
1) $h + k = 3$
2) $h - k = 1$
Adding both equations yields $2h = 4 \implies h = 2$. Substituting back gives $k = 1$. The intersection point is $(2, 1)$.

Step 2: Translate Axis Systems.
Let $x = u + 2$ and $y = v + 1$. Substituting these expressions eliminates the constants: $$\frac{dv}{du} = \frac{(u + 2) + (v + 1) - 3}{(u + 2) - (v + 1) - 1} \implies \frac{dv}{du} = \frac{u + v}{u - v}$$
Step 3: Resolve Homogeneous Equation via $v = zu$.
Substitute $v = zu$ and $\frac{dv}{du} = z + u\frac{dz}{du}$:
$$z + u\frac{dz}{du} = \frac{u + zu}{u - zu} = \frac{1 + z}{1 - z}$$ $$u\frac{dz}{du} = \frac{1 + z}{1 - z} - z = \frac{1 + z^2}{1 - z}$$
Step 4: Separate Variables and Integrate.
$$\left(\frac{1 - z}{1 + z^2}\right) dz = \frac{1}{u} du \implies \int \frac{1}{1 + z^2} dz - \int \frac{z}{1 + z^2} dz = \int \frac{1}{u} du$$ $$\tan^{-1}(z) - \frac{1}{2}\ln(1 + z^2) = \ln|u| + C$$ Using log rules to combine terms and substituting $z = \frac{v}{u}$ simplifies this to:
$$\tan^{-1}\left(\frac{v}{u}\right) - \frac{1}{2}\ln(u^2 + v^2) = C$$
Step 5: Restore Original Coordinates ($u = x - 2$, $v = y - 1$).
$$\tan^{-1}\left(\frac{y - 1}{x - 2}\right) - \frac{1}{2}\ln\left((x - 2)^2 + (y - 1)^2\right) = C$$
